Deklaration und Definition einer Arrays

#define ANZAHL_ELEMENTE Anzahl_Elemente /* Definition der symbolischen Konstanten */
/* Wertebereich des Index ist immer von [0, (Anzahl_Elemente-1)] */
typ arrayname[ANZAHL_ELEMENTE];
typ i;
/* Mit const wird verhindert, dass die Werte eines Feldes verändert werden können. Die Feldgröße wird hier durch den 
Compiler selbst berechnet (Anzahl der Elemente bei der Initialisierung) */
const typ arrayname_const[] = {z1, z..., zn};
/* greift auf das Element mit dem Index x zu und weist den Wert der Variablen i zu */
i = arrayname[x];
/* Beachten Sie, dass Sie niemals einen Zugriff außerhalb des Wertebereichs durchführen.
d.h. x muss immer im Wertebereich [0, ANZAHL_ELEMENTE-1] sein! */

Die Deklaration erfolgt durch die Typangabe der Arraybezeichnung und die ANZAHL_ELEMENTE. ANZAHL_ELEMENTE wird mithilfe von einer symbolischen Konstante festgelegt.

Initialisierung

/* Das Array wird vereinbart. Vom Indexbereich [0, m] werden die 
Werte x1 ... xm zugewiesen und von Index m+1 ... Anzahl_Elemente-1 
wird der Wert 0 zugewiesen. */
typ arrayname_init[ANZAHL_ELEMENTE] = {x1, ... , xm};

Enthält die Initialisierungsliste (gilt also nur bei der Initialisierung der Form typ arrayname[n] = {..., ..., ...};) weniger Elemente als die Dimension des definierten Arrays so werden die Datenelemente von 0 bis (n - 1) mit den angegebenen Werten initialisiert. Die restlichen Elemente werden mit dem Wert 0 initialisiert. Globale und statische Felder werden automatisch mit 0 initialisiert.
