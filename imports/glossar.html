Glossar

Headerfile
(Synonym: Deklarationsdatei)
In der Header-Datei werden alle globalen Variablen (mit extern), Funktionsdeklarationen (Prototypen), Präprozessordirektiven etc. deklariert. (Datei erhält die Extension .h)
Sourcefile
(Synonym: Quelldatei)
  • Im Sourcefile wird der eigentliche Programmcode erfasst. (Datei erhält die Extension .c (C), .cpp beim VC++ Compiler)
  • Werden Programmteile von anderen Modulen benötigt, so werden diese mithilfe der #include-Anweisung eingebunden, insbesondere wird auch das korrespondierende Headerfile eingebunden.
  • Variablendefinitionen
  • Funktionsdefinitionen
  • Lokale Variablendefinitionen und lokale Funktionsdefinitionen (Schlüsselwort static)
Compiler
Ein Compiler (auch Kompiler; von englisch für zusammentragen bzw. lateinisch compilare ‚aufhäufen‘) ist ein Computerprogramm, das Quellcode einer bestimmten Programmiersprache in eine Form übersetzt, die von einem Computer (direkter) ausgeführt werden kann.
Linker
Unter einem Linker oder Binder (auch: „Bindelader“) versteht man ein Computerprogramm, das einzelne Programmmodule zu einem ausführbaren Programm zusammenstellt (verbindet). Auf IBM-Großrechnersystemen wird der Linker linkage editor (englisch) genannt.
Grundprinzip der Datenverarbeitung
"Daten" einlesen > "Daten" verarbeiten > "Daten" ausgeben
Deklaration
Beschreibt / typisiert die Variable, reserviert aber keinen Speicherplatz.
Definition
Es wird für die Variable ein Speicherplatz reserviert bzw. Ergänzung durch Attribute wie const sind möglich.
Initialisierung
Bei der Initialisierung werden - nachdem die Variablen definiert sind - den Variablen gültige Werte zugewiesen. Eine Initalisierung wird i.d.R. nur dann durchgeführt, wenn der Variablen vor der ersten Verwendung im Code kein Wert zugewiesen wird.
Gültigkeitsbereich
Unter Gültigkeitsbereich (=Lebensdauer) versteht man jenen Bereich, in dem die Variable mit ihrem Inhalt existiert und den Wert nicht verliert.
Sichtbarkeit
Unter Sichtbarkeit versteht man jenen Bereich, wo die Werte der Variablen gelesen bzw. verändert werden können.Der Sichtbarkeitsbereich liegt somit immer im Gültigkeitsbereich.
Typecasting
Um Daten verschiedenen Datentyps in einem Ausdruck verarbeiten zu können, besteht die Möglichkeit, dass die Werte in den Variablen in einen anderen Datentyp umgewandelt werden.
