printf() Referenz

%c
 character 
%d
 decimal (integer) number (base 10) 
%e
 exponential floating-point number 
%f
 floating-point number 
%i
 integer (base 10) 
%o
 octal number (base 8) 
%s
 a string of characters 
%u
 unsigned decimal (integer) number 
%x
 number in hexadecimal (base 16) 
%%
 print a percent sign 
\%
 print a percent sign

\r

\t

\v

\\
